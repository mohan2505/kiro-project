# HelpWave 🚨

**A Simple Emergency Alert Web Application for Personal Safety**

HelpWave is a lightweight, browser-based emergency alert system designed to help individuals quickly attract attention in unsafe situations. With a single click, it activates visual and audio alerts to draw attention and signal for help.

## 🎯 Purpose

In situations where someone feels unsafe or threatened, HelpWave provides an immediate way to:
- Draw attention through loud audio alerts
- Create visual alerts with screen flashing
- Signal distress without needing to make a phone call
- Work offline once loaded

This tool is particularly designed with women's safety in mind, offering a quick and accessible way to attract help when needed.

## ✨ Features

- **Large Help Button**: Easy to activate even in stressful situations
- **Visual Alert**: Screen flashes red to attract visual attention
- **Audio Alert**: Continuous loud siren sound to attract auditory attention
- **Emergency Message**: Clear on-screen message indicating help is needed
- **Stop Control**: Simple button to deactivate the alert when safe
- **Keyboard Shortcuts**: Space/Enter to activate, Escape to stop
- **Offline Capable**: Works without internet connection after initial load
- **Responsive Design**: Works on both mobile and desktop devices

## 🚀 How to Use

1. **Open the App**: Load `index.html` in any modern web browser
2. **Activate Alert**: Click the large red "HELP!" button
3. **Alert Active**: The screen will flash red and a loud siren will play
4. **Stop Alert**: Click the green "STOP ALERT" button when safe

### Keyboard Shortcuts
- **Space** or **Enter**: Activate alert
- **Escape**: Stop alert
